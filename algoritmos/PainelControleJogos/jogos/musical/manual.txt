O QUE É O MELODY MASTER?

Software de educação musical que utiliza o alto-falante do sistema para emitir sons e testar a habilidade do usuário em reconhecer diferentes intervalos melódicos.                                 


SOBRE COMO OPERAR COM O PROGRAMA

Este programa está estruturado em 7 níveis de dificuldade. Cada nível, possui 10 questões. Cada pergunta consiste em um intervalo melódico gerado aleatoriamente que você deve identificar conforme as opções dadas. Para responder a uma pergunta, digite o caractere entre "<>" correspondente à opção desejada e tecle  <ENTER>.

     
UM EXEMPLO DE PERGUNTA
    
PERGUNTA 5:
Nota 1 (VOCÊ VAI OUVIR A NOTA) - Nota 2 (VOCÊ VAI OUVIR A NOTA)
O intervalo entre as notas eh uma segunda Menor<E> ou uma segunda Maior<M>?
[Repetir<R>]
Digite uma resposta e <Enter>


PONTUAÇÃO E NÍVEIS

- Enquanto você responde, seu numero de acertos é contabilizado
- Para passar de nível são necessários pelo menos 7 acertos
- Uma vez ultrapassado um nível, seu acesso será liberado aos níveis anteriores
- Se, ao refazer um nível, seu escore for melhor que o de alguma execução anterior, sua nova pontuação será gravada


SOBRE OS ARQUIVOS USADOS POR ESTE PROGRAMA
    
Um arquivo com o seu ‘USERNAME + extensão .DAT’ foi criado no mesmo diretório de onde você executou este programa. Nele ficarão armazenados os dados a respeito do seu desempenho. Não delete nem edite o arquivo fora deste programa ou esses dados serão perdidos.                                                                     
    
Se desejar mover o arquivo executável para outro diretório, mova o arquivo “SeuNomeAqui.dat” junto com ele.      


SOBRE O CONTEÚDO DOS NÍVEIS
    
Um pouco de teoria musical: 
    
A escala diatônica tem 7 notas:
C, D, E, F, G, A, B      
    
Entre estas notas, temos 12 semitons:
C, C#, D, D#, E, F, F#, G, G#, A, A#, B
    
O que este software faz é gerar randomicamente intervalos ascendentes e descendentes conforme grupos preestabelecidos (veja o quadro).
